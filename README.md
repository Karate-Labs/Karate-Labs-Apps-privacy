# Karate-Labs-Apps-privacy

本アプリは、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本ポリシーでは、本アプリにおけるデータの取り扱いについて説明します。

1. 収集する情報

本アプリは、ユーザーの個人情報を一切収集しません。
氏名・住所・メールアドレス・写真・連絡先などは取得しません。

2. アプリ内データの扱い

本アプリに入力したメモ・記録・設定値などはすべてユーザーの端末内に保存されます。
外部サーバーへ送信することはありません。

3. カメラ権限について（必要な場合のみ）

本アプリは、計測や動作記録のためにカメラ権限を使用する場合があります。

カメラ映像は端末内のみで処理され、外部に送信されません。

録画データを自動アップロードしたり、第三者に共有することはありません。

4. データの第三者提供

本アプリは、ユーザーのデータを第三者へ提供することはありません。

5. 広告・解析ツールの利用

本アプリは広告配信および解析ツールを使用していません。
（使用する場合は、そのツール名と目的を追記します。）

6. 未成年の利用について

本アプリは、子どもを含むすべてのユーザーが利用できるアプリです。特別な年齢情報を取得することはありません。

7. プライバシーポリシーの変更

必要に応じて本ポリシーの内容を変更する場合があります。変更後の内容はこのページで公開されます。
